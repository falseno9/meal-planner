# Meal Planner

Local dinner planner for maintaining a meal-item database, generating balanced weekly plans, reserving eating-out slots, and aggregating groceries from entered ingredients.

## Run

```bash
npm install
npm run dev
```

Open the Vite URL shown in the terminal, usually `http://localhost:5173`.

## Data

Local data is stored as JSON files:

- `data/categories.json`
- `data/meal-items.json`
- `data/plans.json`

## Checks

```bash
npm test
npm run build
```
